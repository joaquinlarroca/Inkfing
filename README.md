## What is this project?

Welcome to InkFing! This project is a simple extension that allows Fing (Facultad de Ingeniería, UdelaR) students to give their day to day web apps a refreshed dark-mode.
It's designed to help incoming students find study partners and collaborative projects to work on.

### InkFing currently supports dark-mode on this sites

- [https://bedelias.udelar.edu.uy/](https://bedelias.udelar.edu.uy/)

### Planned dark-mode support

- [https://eva.fing.edu.uy/](https://eva.fing.edu.uy/)
- [https://www.fing.edu.uy/](https://www.fing.edu.uy/)

### Tech Stack

- HTML
- CSS
- JS
- Manifest V3

## Quick Start

### Option 1: Load a temporary addon/extension

- Firefox: [https://developer.mozilla.org/en-US/docs/Mozilla/Add-ons/WebExtensions/Your_first_WebExtension#installing](https://developer.mozilla.org/en-US/docs/Mozilla/Add-ons/WebExtensions/Your_first_WebExtension#installing)

- Chrome: [https://developer.chrome.com/docs/extensions/get-started/tutorial/hello-world#load-unpacked](https://developer.chrome.com/docs/extensions/get-started/tutorial/hello-world#load-unpacked)

### Option 2: Installing from file

- Firefox
    1. goTo `about:addons`
    2. Click on the button with a cogwheel icon `Install for all Add-ons`
    3. Select `Install Add-on From File...`
    4. Upload the zipped extension

#### Zipped extension/addon structure

```txt
InkFing_vX.x
├── css
│   *.css
├── icons
│   *.png
├── manifest.json
```
